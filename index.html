Research overview

Evaluation of Explainable AI for Time Series

My PhD research focuses on evaluating explanations of AI decisions, primarily in the context of time series. This page collects various projects that I have worked on in my PhD, including computational evaluation of explainable AI (XAI) explanations, and more human-centric evaluation.

Perturbation metrics

Class-dependent evaluation effects

Perturbation analysis is often used to evaluate how "correct" explanations are. In the below projects, we studied when that evaluation depends on the predicted class, and what that means for trusting the perturbation metrics. Here, we applied evaluation metrics to feature attribution explanations, which compute importance scores to input features (time points) for a specific prediction.

Paper 1

Class-dependent perturbation effects in evaluating time-series attributions

Takeaway: Perturbation-based evaluation metrics can behave very differently across classes, even for the same dataset and model. This implies strong aggregate scores can reflect class-specific model behavior rather than better attribution quality.

The paper evaluates feature attribution explanation methods across datasets, model architectures, and perturbation strategies in the context of time series classification. It finds that perturbation metrics can be sensitive for some classes and much less sensitive for others, with the strongest perturbation strategies often showing the largest class differences.

Baer, G., Grau, I., Zhang, C., Van Gorp, P. (2025). Class-Dependent Perturbation Effects in Evaluating Time Series Attributions. In: Guidotti, R., Schmid, U., Longo, L. (eds) Explainable Artificial Intelligence. xAI 2025. Communications in Computer and Information Science, vol 2579. Springer, Cham. https://doi.org/10.1007/978-3-032-08330-2_14

Paper 2

Why do class-dependent evaluation effects occur?

Takeaway: Class-dependent evaluation effects can come from simple differences between class features, such as amplitude or duration. Additionally, perturbation metrics can disagree with ground-truth metrics, so you may want to be careful in making comparisons.

The paper uses synthetic time series with known relevant feature locations to study when class-dependent evaluation effects occur. By varying feature types and class contrasts, it shows that these effects can appear even in simple controlled settings.

The results show that perturbation-based evaluation metrics and ground-truth-based metrics often provide different views of attribution quality across classes.

Baer, G., Grau, I., Zhang, C., Van Gorp, P. (2026). Why do Class-Dependent Evaluation Effects Occur with Time Series Feature Attributions? A Synthetic Data Investigation. In: Koprinska, I., Mendes-Moreira, J., Branco, P. (eds) Machine Learning and Principles and Practice of Knowledge Discovery in Databases. ECML PKDD 2025. Communications in Computer and Information Science, vol 2842. Springer, Cham. https://doi.org/10.1007/978-3-032-19105-2_27

Human-centric evaluation

Computational metrics and human understanding

A computational evaluation metric is useful only if we know what it tells us. This project studies how computational explanation correctness relates to whether people can understand AI decision patterns.

User study

Linking Computational XAI Evaluation to Human Understanding

Takeaway: Explanation correctness can affect human understanding, but the relationship is not always straightforward. Incorrect explanations can hurt understanding, while correct ones do not guarantee that people learn (and understand) the model's decision pattern.

In a user study, participants used explanations to predict an AI system's decisions in a time-series classification task. This task is called forward simulation and can be seen as a proxy measure for understanding. The study controlled explanation correctness at four levels and found that understanding dropped at 70% and 55% correctness compared with fully correct explanations, but did not keep dropping below 70%. 100% and 85% correct explanations are statistically indistinguishable in terms of forward simulation accuracy.

The results suggest that computational XAI evaluation metrics should not be treated as direct proxies for human understanding. Correctness can help, but people differ in whether they learn from explanations, and self-reported understanding does not always match actual task performance.

Baer, G., Zhang, C., Grau, I., & Van Gorp, P. (2026). Does Explanation Correctness Matter? Linking Computational XAI Evaluation to Human Understanding. https://arxiv.org/abs/2603.25251v1

Synthetic data

Tools for controlled evaluation

xaitimesynth is a small Python package for synthetic time-series XAI evaluation. It creates datasets with known feature locations, so attribution methods can be tested in controlled and reproducible settings.

Package

xaitimesynth: A Python Package for Evaluating Attribution Methods

Takeaway: xaitimesynth makes it easier to evaluate time-series attribution methods with synthetic ground truth, instead of rebuilding custom data generators for each study.

The package generates univariate and multivariate time series by combining a background signal with localized, class-discriminating features. It tracks the feature locations as ground-truth masks, which can then be used to test whether attribution methods highlight the right time points.

It supports flexible dataset definitions through a fluent Python API with method chaining and YAML configuration files, and includes standard localization metrics such as AUC-PR, AUC-ROC, Relevance Mass Accuracy, and Relevance Rank Accuracy.
